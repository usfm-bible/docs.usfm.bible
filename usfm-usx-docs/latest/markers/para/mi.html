mi

Summary

Description

Regular paragraph.

  • See also: pmo, pmc

Syntax

  • USFM: \mi_text...

  • USX: <para style="mi">text...</para>

Added

1.0

Properties

StyleType

Paragraph

OccursUnder

[ChapterContent], [Sidebar]

TextType

VerseText

TextProperties

paragraph, publishable, vernacular

Structure

Examples

each other,
\b
\q1
\v 17 “We played the flute,
\q2 but you would not dance!
\q1 We sang a funeral song,
\q2 but you would not mourn!”
